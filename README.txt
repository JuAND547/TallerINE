JUEGO DE REFLEJOS "COLOR REACTOR"

Pasos para ejecutar el código:

1- Asegurarse de tener las librerías que aparecen en el comienzo del código.
2- Llenar las variables "ssid" y "password" por la red Wi-Fi y la contraseña que usarás, respectivamente.
3- Ajustar los pines/conexiones.
4- Ejecutar código en ESP32.
5- ¡Jugar!

Modificaciones adicionales:

- Cantidad de LEDs y botones (default: 5)
- Tiempo de juego (primer valor en segundos, default: 60)
- Tiempo de reacción: Tiempo máximo que tiene el jugador para presionar el botón correcto una vez este se prende. (En milisegundos. Default: 2000)
- Sonido (variable sonidoactivado, true=ON, false=OFF)
